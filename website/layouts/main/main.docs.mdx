import { Typography } from "@nitsan770/portfolio.typography.typography";

A component that acts as a layout component and positions the header, footer and content.
Use with `ThemeProvider` to enforce themeing.

### Component usage

```js
<Main>{children}</Main>
```

### Render sample main layout.

```js live
<MemoryRouter>
  <Main>
    <Typography variant="h1">This is the body.</Typography>
  </Main>
</MemoryRouter>
```
